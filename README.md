# Reportes (MVP) – Sprint (Committed)

**Sprint:** 02/02/2026 – 07/02/2026  
**Tiempo calendario:** 12 h (2 h/día)  
**Capacidad efectiva (80%):** 9.6 h (se deja 20% para buffer)

## Objetivo del Sprint
Entregar un MVP en React que permita **crear** reportes y **consultarlos** (listado + detalle), con persistencia en **localStorage**.

## Funcionalidades (Committed)
- Crear reporte (tipo, descripción, ubicación) con validación
- Guardado y lectura desde localStorage
- Listado + detalle
- Seed de 1 reporte demo (si está vacío)

## Cómo correr el proyecto
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Ejecuta en modo desarrollo:
   ```bash
   npm run dev
   ```
3. Abre la URL que te muestre la terminal (normalmente `http://localhost:5173`).

## Subida a GitHub (rápido)
- Crea repo y haz commits.
- Pega el link del repo en el Word (sección “Acceso a GitHub”).
